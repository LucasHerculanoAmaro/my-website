Mtech

Este projeto foi feito durante as aulas da Generation Brasil, sendo desenvolvido por um grupo de estudantes e apresentado para empresas parceiras do programa.

O objetivo é criar uma rede social para ajudar mulheres a ingressar na área de Tecnologia.

O projeto MTech conta com tela de Login, tela de Cadastro e tela de Início, e os usuários podem criar Temas e contam com ações para criar, ler, atualizar e deletar uma postagens.

Lista de Funcionários

Este projeto foi feito com base no conteúdo do canal Java Guides, e o objetivo é criar um projeto Full Stack para controle de funcionários.

Ao implementar o método CRUD, é possível cadastrar um funcionário, atualizar suas informações, deletar os dados do funcionário e ver a lista de funcionários cadastrados no banco de dados.

O projeto Lista de Funcionários conta com uma tela de cadastro de funcionários, tela para atualizar registros, tela para deletar um funcionário e uma tela para apresentar uma lista com todos os funcionários.

Abaixo estão as linguagens e ferramentas necessárias para desenolver este projeto:

  • Springboot;
  • MySQL;
  • Angular;
  • Node.js;
  • BootStrap;
  • Postman;
